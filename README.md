# Restaurants

## Project setup
```
npm install
```

### Enviroment Settings
Create a .env.local file on the app root and set yours API keys variables VUE_APP_YELP_API_KEY and VUE_APP_GMAP_API_KEY.
Example:
```
VUE_APP_YELP_API_KEY=abc123
VUE_APP_GMAP_API_KEY=def456
```
